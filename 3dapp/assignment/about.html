About This Project

An immersive Web 3D application showcasing interactive smart home devices

Project Overview

Smart Living 3D is an interactive web application designed to showcase the integration of 3D models with web technologies. This project demonstrates how Three.js can be used to create engaging user experiences that allow users to interact with virtual representations of smart home devices.

The application features three detailed 3D models: a smart TV, a smart refrigerator, and a smart lamp. Each model can be manipulated and controlled through an intuitive user interface, demonstrating real-world functionalities such as turning devices on/off, adjusting settings, and changing visual properties.

Smart TV

Interactive TV with power controls, channel switching, and volume adjustment

Smart Refrigerator

Temperature controls, door animation, and customisable appearance

Smart Lamp

Light controls, brightness adjustment, and color temperature settings

3D Visualisation

Interactive camera controls, wireframe mode, and lighting adjustments

Development Process

3D Modeling

All 3D models were created using Blender. The modeling process focused on achieving a balance between visual quality and performance. Each model was optimised with efficient geometry, appropriate materials, and careful UV mapping for textures.

Model Optimisation

Each model was carefully optimised to ensure efficient rendering in a web environment:

  • Reduced polygon count while maintaining visual fidelity
  • Optimised texture sizes and compression
  • Created custom LOD (Level of Detail) systems
  • Implemented efficient UV mapping

Animation Rigging

The models include interactive elements that required careful rigging:

  • Fridge door opening and closing animations
  • TV screen display state changes
  • Lamp lighting effects and brightness adjustments
  • Smooth transitions between different states

Web Implementation

The web application was built using HTML5, CSS3, and JavaScript. Three.js was used as the primary library for 3D rendering and interaction. The UI was designed with Bootstrap to ensure responsiveness across different devices.

Key technical features implemented include:

  • Custom model loading and positioning
  • Interactive controls for each model
  • Dynamic lighting with adjustable properties
  • Wireframe mode for examining model geometry
  • Camera position controls for different viewpoints
  • Real-time model statistics display
  • Custom shaders for realistic materials
  • Post-processing effects for enhanced visuals
  • Database integration for state persistence

Challenge: Model Loading and Placement

One significant challenge was ensuring proper model loading and placement within the scene. This was addressed by implementing a bounding box calculation system that automatically scales and positions each model based on its dimensions, ensuring consistent presentation regardless of the original model size.

Challenge: Interactive Controls

Creating model-specific interactive controls required careful traversal of the model hierarchy to identify and manipulate specific mesh components. This was solved by using naming conventions within the models and implementing targeted material and transformation modifications.

Deeper Understanding Implementation

This project implements several advanced techniques and architectural patterns that go beyond the basic requirements to demonstrate a deeper understanding of Web 3D application development:

MVC Architecture

Implemented the Model-View-Controller pattern to separate data, presentation, and business logic:

  • Model: Manages device states and data persistence
  • View: Handles UI rendering and user feedback
  • Controller: Coordinates interactions between Model and View

Database Integration

Created a SQLite database system with PHP backend for:

  • User preference storage and retrieval
  • Interaction analytics and tracking
  • Model state persistence across sessions
  • Statistical data collection for usage patterns

Custom GLSL Shaders

Developed custom shader materials for enhanced visual quality:

  • Glass shader with fresnel effect for TV screen
  • Metal shader with realistic reflections
  • Emissive shader with pulsing glow effects
  • Toon shader for stylized rendering options

Post-Processing Pipeline

Implemented advanced post-processing effects:

  • Bloom effect for emissive materials
  • Ambient occlusion for realistic shadows
  • Film grain for stylized rendering
  • Color correction for enhanced visual quality

These advanced implementations demonstrate a comprehensive understanding of modern web 3D development techniques and architectural patterns. The modular design allows for easy expansion and maintenance, while the custom shaders and post-processing effects provide enhanced visual quality beyond standard Three.js materials.
